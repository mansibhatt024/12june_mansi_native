Define the terms: Website, Webpage, Web browser, Web server, HTML, CSS 1.Website: -A website is a collection of related web pages, multimedia content, and other digital assets that are typically identified with a common domain name and are published on at least one web server. Websites are accessed and viewed using web browsers. 2.Webpage: -A webpage is a single document or a resource of information that is suitable for the World Wide Web and can be accessed through a web browser. It can contain text, images, multimedia elements, hyperlinks, and other types of content. Webpages are the building blocks of websites. 3.Web Browser: -A web browser is a software application used to access information on the World Wide Web. It interprets HTML code, displays text and multimedia content, and allows users to navigate between webpages by clicking on hyperlinks. Popular web browsers include Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge. 4.Web Server: -A web server is a computer program or a device that hosts websites and delivers web content to users over the internet. When a user requests a webpage, the web server processes the request and sends the requested files (such as HTML documents, images, CSS files, etc.) to the user's web browser, which then displays the webpage. 5.HTML (Hypertext Markup Language): -HTML is the standard markup language used to create webpages. It defines the structure and layout of a webpage by using a variety of tags and attributes. HTML elements are represented by tags, which indicate how content should be displayed on a webpage. Web browsers interpret HTML code to render webpages for users. 6.CSS (Cascading Style Sheets): -CSS is a stylesheet language used for describing the presentation and formatting of a document written in HTML. CSS controls the layout, colors, fonts, and other visual aspects of a webpage, allowing web developers to style web content and make it visually appealing. CSS rules can be applied internally within an HTML document or externally in separate CSS files.
